# Admin Dashboard Component Map

Date: 2026-07-09

This document maps the admin dashboard areas to backend functions, views, tables, and fields that already exist in the repository documentation.

| Dashboard widget name | Backend function or view used | Payload fields used | What the widget should display | Type |
| --- | --- | --- | --- | --- |
| Warning Center | `get_dashboard_payload_state`, `get_warning_center_counts_state`, `get_warning_center_detail_state`, `v_warning_center_critical_items`, `v_warning_center_warning_items`, `v_warning_center_review_items` | `payload.warning_center_counts.critical_count`, `warning_count`, `review_count`; `payload.warning_center_detail.critical[]`, `warning[]`, `review_needed[]`; `item_type`, `source_id`, `reservation_id`, `vehicle_id`, `risk_level`, `source_status`, `expected_return_snapshot`, `contract_period_id`, `reminder_state`, `message` | Critical, warning, and review-needed alerts derived from reservation, vehicle, and case state. | alert |
